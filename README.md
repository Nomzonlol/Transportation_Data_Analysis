# Transportation Data Analysis Project

### Project Summary
This project analyzes transportation trends using the **NHTS** (National Household Travel Survey) dataset and driving behavior using the **NGSIM** (Next Generation Simulation) dataset. As an Engineering Intern, I have organized and visualized this data to support informed decision-making for the transportation planning group.

### User Guide
To run the code and generate the solutions, follow these sequential steps in the Jupyter Notebook:
* **Load Libraries:** Run the first cell to import `pandas`, `matplotlib`, and `seaborn`.
* **Load Data:** Ensure the `NHTS.csv` and `NGSIM.csv` files are in the same folder as the notebook. Run the data loading cells.
* **Execute NHTS Visualizations:** Run the cells for the Bar Chart, Histogram, and Boxplot to see national vehicle trends.
* **Execute NGSIM Visualizations:** Run the cells for the Time-Series plots. Note: The data is filtered by vehicle trajectory and sorted by time to show accurate driving behavior.
* **Review Insights:** Read the markdown cells below each graph for professional engineering analysis.
